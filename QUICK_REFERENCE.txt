MAIN USAGE
================================================================================

python main.py

# With custom files
python main.py --profiles data/input/my_profiles.csv \
               --activities data/input/my_activities.csv

# With Hungarian assignment
python main.py --assignment --assignment-method hungarian

# Use variant proximity formula
python main.py --proximity-formula variant

================================================================================
CONFIGURATION OPTIONS
================================================================================

Edit config.json to customize:

- threshold: Skill level threshold (default: 3.0)
- proximity_formula: "standard" or "variant"
- weight_strategy: "uniform", "requirement_based", or "custom"
- enable_optimal_assignment: true/false
- assignment_method: "auto", "hungarian", or "greedy"

================================================================================
PERFORMANCE TIPS
================================================================================

For Speed:
  - Use standard proximity formula
  - Skip visualizations
  - Use greedy assignment for large datasets

For Memory:
  - Reduce number of skills
  - Use greedy instead of Hungarian
  - Close other applications

================================================================================
TROUBLESHOOTING
================================================================================

Out of Memory:
  - Reduce dataset size
  - Use fewer skills
  - Increase system RAM

Too Slow:
  - Use standard formula (--proximity-formula standard)
  - Skip assignment (--no-assignment)
  - Use greedy method (--assignment-method greedy)

Import Errors:
  - Ensure you're in project root directory
  - Check Python path includes src/

Visualization Errors:
  - Install matplotlib and seaborn
  - Or skip visualizations (disabled by default)
